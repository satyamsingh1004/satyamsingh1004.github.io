Syntax: Defaults(DataSource) This function returns a record that contains the default values for the data source. Example: 1. A Datasource with 2 columns Country - (Default Value - India) City - (Default Value - Delhi) Defaults(Datasource) will return a record {Country: "India", City: "Delhi"} If a column within the data source doesn't have a default value, that property won't be present. Exmaple: 2. A Datasource with 2 columns Country - (Default Value - India) City - Defaults(Datasource) will return a record {Country: "India"} When you work with a collection or another data source that doesn't support default values, the Defaults function will return an empty record. 3. A Datasource with 2 columns Country - City - Defaults(Datasource) will return a record {} Reference: https://docs.microsoft.com/en-us/powerapps/maker/canvas-apps/functions/function-defaults
